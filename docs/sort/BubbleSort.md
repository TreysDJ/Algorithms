## Bubble Sort
Самая простая сортировка. Сравнивает первый элемент со вторым, второй с третьим, третий с четвёртым и так далее. Если какой то элемент больше другого они меняются местами
```java
public void sort(int[] arr) {
        boolean isSorted = false; // boolean переменная для отслеживания, сортируется ли массив ещё раз. Если нет - останавливаем цикл while, а за ним закрывается и цикл for
        while (!isSorted) { 
            isSorted = true; // даём isSort true чтобы если условие не прошло, цикл завершился
            for (int i = 1; i < arr.length; i++) {
                if (arr[i] < arr[i - 1]) { // сравниваем 0 с 1, 1 с 2 и т.д. Если i < i-1 - меняем их местами и даём isSort значение false, чтобы цикл сработал ещё раз
                    int currentIndex = arr[i];
                    arr[i] = arr[i - 1];
                    arr[i - 1] = currentIndex;
                    isSorted = false;
                }
            }
        }
    }
```
Лучший случай - если все элементы в массиве уже будут отсортированы. Сложность O(n), так как циклу for всё равно придётся пройтись по каждому элементу массива, чтобы завершить его.
Худший случай - если все элементы будут не отсортированы. Сложность O(n^2), так как цикл for будет повторяться, пока все элементы не будут отсортированы. Алгоритм проходит по массиву вложенными циклами: внешний цикл - n раз, внутренний - столько раз, пока массив не будет отсортирован, и среднее количество сравнений остаётся порядка n^2.
